# What's this?

Next.js and Material UI boilerplate.

Using [TMDB people API](https://developers.themoviedb.org/3/getting-started) as example of data fetching.

# Environment

Node.js 16.x

# Command List

## Install Packages

```sh
$ yarn
```

## Start dev

```sh
$ yarn dev
```

## Build

```sh
$ yarn build
```

## StoryBook

```sh
$ yarn story
```
